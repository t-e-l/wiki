# key-combos

_Do things even quicker with key-combos!_

the use of key combos generally requires USE_TMUX=true in ~/.tel/configs/userprefs.sh 

These shortcuts will open and run in a new window split:
TEL + s 	search
TEL + p		dialer
TEL + a		apps
TEL + e		edit
TEL + o		open
TEL + c		calculator
TEL + b		bookmarks
TEL + w		wifi
